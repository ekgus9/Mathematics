# 벡터가 직교한다



벡터 공간에서도 피타고라스의 정리는 적용된다. 따라서 벡터 x와 y가 직교한다면 



    ||x||^2 + ||y||^2 = ||x+y||^2
    xy = 0
  
  
  
**즉, x와 y의 내적값이 0이라면 벡터는 직교한다.**
