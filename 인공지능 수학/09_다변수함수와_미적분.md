### 9-2 벡터와 공간 기하



* 사잇각 : 벡터 x, y에 대해 x * y = ||x|| ||y|| cosθ인 θ가 존재하며, 이 θ를 x와 y가 이루는 각 (C-S 부등식 : |x * y| <= ||x|| ||y||)

```
b.dot_product(a) # 내적
b.cross_product(a) # 외적
arccos(a.dot_product(a) / (a.norm()*b.norm())) # 사잇각
```
